# Typengine

A lightweight game engine written in C++. It features Vulkan for rendering, Duktape for Javascript scripting, and a custom UI system.

## Features
- **Vulkan Renderer**: Fast and modern rendering backend.
- **Scripting**: Integrated Duktape for JS scripting.
- **UI System**: Built-in UI elements and rendering.
